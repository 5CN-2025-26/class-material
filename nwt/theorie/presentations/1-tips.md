
## Präsentationstechnik – Feedback des Professors

### Häufige Fehler
| Fehler | Beispiel |
|-------|---------|
| **Füllwörter** | „äh“, „okay“, „so halt“, „genau“ |
| **Schnelles Sprechen** | → Zuhörer können nicht folgen |
| **Beamer fixieren** | → wirkt unvorbereitet |
| **Keine Vorstellung** | → fehlende Professionalität |
| **Zeitüberschreitung** | → Prüfungsordnung zwingt zum Abbruch |

### Tipps für Reifeprüfung
1. **Handout vorher verteilen** → Zuhörer können **folgen**, nicht nur mitschreiben
2. **Zeitmanagement:**
   - **Üben** mit Stoppuhr
   - **Mitte der Präsentation auf Uhr schauen**
   - **Nicht schneller sprechen** → lieber Inhalt kürzen
3. **Blickkontakt:**
   - **Schweifen lassen**
   - **Nicht nur einen anschauen**
   - **Nicht zur Decke starren**
4. **Ende klar signalisieren:**
   - Kein „ich hab nur noch 3 Folien“
   - **„Vielen Dank für Ihre Aufmerksamkeit“** → Ende

### Handout-Nutzung
- **Ziel:** Zuhörer sollen **mithören**, nicht nur mitschreiben
- **Ideal:** Handout **vorher überflogen** → nur **Ergänzungen notieren**
